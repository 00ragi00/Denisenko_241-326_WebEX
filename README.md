# inguaSchool - Education Platform

Веб-платформа для записи на курсы и к репетиторам.

## Функционал

- **Каталог курсов** - просмотр доступных курсов с фильтрацией по предметам и уровням
- **База репетиторов** - поиск преподавателей с рейтингами и отзывами
- **Личный кабинет** - управление заявками и профилем пользователя
- **Система заявок** - онлайн-запись на курсы и к репетиторам

## Технологии

- HTML5 / CSS3
- JavaScript
- Bootstrap 5
- REST API

## Структура

```
├── index.html          # Главная страница
├── account.html        # Личный кабинет
├── styles/
│   └── style.css       # Стили
└── scripts/
    ├── app.js          # Основная логика
    ├── api.js          # Работа с API
    └── price.js        # Расчёт стоимости
```

## Запуск

Откройте `index.html` в браузере
